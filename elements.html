Block-level elements in HTML

  1. table tag used to create a table
  2. ol tag used to create a ordered list
  3. ul tag used to create a unordered list
  4. p tag used to write a paragraph
  5. li tag used to create a list
  6. div tag defines a division or a section in an HTML document.
  7. nav tag used to create a navigation bar
  8. main tag specifies the main content of a document.
  9. dt tag defines a term/name in a description list.
  10. figcaption tag defines a caption for a figure element.
  11. figure tag specifies self-contained content, like illustrations, diagrams, photos, code listings, etc.

Inline Elements

  1. span tag is an inline container used to mark up a part of a text, or a part of a document.
  2. a tag defines a hyperlink, which is used to link from one page to another.
  3. abbr tag defines an abbreviation or an acronym, like "HTML", "CSS", "Mr.", "Dr.", "ASAP", "ATM".
  4. b tag used to bold the text
  5. br tag inserts a single line break.
  6. input tag used to get the input from the user
  7. textarea tag defines a multi-line text input control.
  8. img tag used to embed an image to the html page
  9. select element is used to create a drop-down list.
  10. time tag defines a specific time
